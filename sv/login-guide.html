Steg-för-steg-guide

Första inloggningen efter ändringen

Vid din första inloggning efter att ditt användarnamn har ändrats måste du använda "Annan användare" på Windows-inloggningsskärmen — inte din befintliga profil. Den här guiden visar dig hur med skärmbilder.

Ditt nya inloggningsnamn
firstname.lastname@ad.kry.se
↓
firstname.lastname@kry.se

Lösenordet förblir detsamma.

⚠️

Tryck inte på din befintliga profil. Om du gör det misslyckas inloggningen. Använd alltid "Annan användare" längst ner till vänster på inloggningsskärmen vid din första inloggning.

1

Känn till go-live-datumet

  1. Sverige: måndag 1 juni 2026.
  2. Du kommer att veta ditt nya användarnamn i förväg — det blir firstname.lastname@kry.se.
  3. Ditt lösenord förblir helt oförändrat.
  4. Datumet är utlösaren. Du kommer att få påminnelser via e-post i förväg — men du behöver inte vänta på dem.
2

Logga ut — endast om du fortfarande är inloggad

Hoppa över det här steget om din dator redan är på Windows-inloggningsskärmen. Gå direkt till Steg 3.

  1. Klicka på Windows (Start)-knappen (längst ner till vänster på skärmen).
  2. Klicka på ditt profilnamn/ikon överst i Start-menyn.
  3. Välj "Logga ut".
  4. Vänta tills Windows-inloggningsskärmen visas.
Start-meny → klicka på profil → Logga ut
3

Logga in med ditt nya användarnamn

  1. På inloggningsskärmen, titta längst ner till vänster.
  2. Klicka på "Annan användare" (detta är det viktiga steget — klicka inte på din befintliga profil ovanför).
  3. I fältet E-postadress, ange ditt nya användarnamn: firstname.lastname@kry.se
  4. Ange ditt befintliga lösenord.
  5. Tryck Enter för att logga in.
Inloggningsskärm → klicka på Annan användare längst ner till vänster

Ange sedan dina nya uppgifter

Efter att du klickat på "Annan användare" visas denna skärm. Ange ditt nya användarnamn och ditt befintliga lösenord.

Ange firstname.lastname@kry.se + ditt befintliga lösenord
FELSÖKNING

Om något går fel

Jag klickade på min gamla profil av misstag och inloggningen misslyckades
Inga problem — detta är förväntat om du klickar på den gamla profilen. Logga ut (eller vänta tills inloggningen misslyckas), och börja om från Steg 2 ovan. Se till denna gång att klicka på "Annan användare" längst ner till vänster.
Jag kommer inte ihåg mitt lösenord
Använd aka.ms/sspr från en annan enhet (telefon, surfplatta, en annan dator) för att återställa ditt lösenord via Authenticator-appen eller en reservmetod. Kom sedan tillbaka och logga in med ditt nya användarnamn och det nya lösenordet.
Skärmen ser annorlunda ut än dessa skärmbilder
Windows uppdaterar då och då inloggningsupplevelsen. Det viktiga steget är detsamma: leta efter "Annan användare" längst ner till vänster på inloggningsskärmen och använd det för att logga in med ditt nya användarnamn.
Jag har fortfarande problem
Skapa ett Zendesk-ärende på ask-it.kry.se.
